# README #



### How do I get set up? ###
Preerquisites:
* Java 8  
* Maven 3  
* Git  


Its a simple maven based spring boot project.

Or else.
clone the project in an empty folder.   
* git clone -b springdoc-openapi-intro https://github.com/teq-niq/sample.git  
* cd sample  
* mvn clean package  
* java -jar target/sample-0.0.1.jar
